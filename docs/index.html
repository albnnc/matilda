    Usage

    To begin with you need to install Matilda. It was released in npm under @albnnc/matilda package. If you want you can use jsDelivr CDN, go on right to the download page. You can also try Matilda and copy-paste the stylesheet into your before all other stylesheets.

    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@albnnc/matilda/dist/matilda.css" />

    But if you want to customize it (e.g. change color palette) you need to install it and compile manually. Simply install it.

    npm install --save @albnnc/matilda

    And import in your SASS code.

    @import '@albnnc/matilda/scss/matilda'
    $placeholder-shown-selector: ':placeholder-shown';
$enable-ripple: true !default;
$font-size-xs: ($font-size-base * 0.75) !default;
$table-font: $font-weight-normal $font-size-sm $font-family-base !default;
$table-head-font: $font-weight-bold $font-size-xs $font-family-base !default;
$btn-text-transform: uppercase;
$form-float-above-top: 0.55rem;
$form-float-container-padding-y: 1.2rem;
$form-float-container-padding-x: 0.7rem;
$form-float-container-filled-bg: rgba(0, 0, 0, 0.05);
$code-bg: $light !default;
$code-padding: 0 0.2rem !default;
$code-border-radius: $border-radius !default;
$pre-bg: $light;
$pre-code-padding: 1rem !default;
$pre-code-border-radius: 0 !default;

    Buttons

    General Bootstrap buttons can be used with their common classes. In addition there is .btn-contained-* classes to make Material contained buttons. Buttons have the ripple effect and you can also use .ripple class and ripple($color: $black) mixin in your SASS code. If you need to make raised button, then you should use a contained one and apply .elevation-* class to it.

    
<button class="m-1 btn btn-primary">
  Button
</button>
<button class="m-1 btn btn-outline-primary">
  Outline
</button>
<button class="m-1 btn btn-contained-primary">
  Contained
</button>
<button class="m-1 btn btn-contained-primary elevation-2">
  Raised
</button>

    It's easy to reach many Material Design variations of buttons with Bootstrap utility classes.

    
<div class="m-1 btn-group bg-white">
  <button class="p-2 btn btn-outline-dark">
    <i class="material-icons align-middle">
      format_align_left
    </i>
  </button>
  <button class="p-2 btn btn-outline-dark border-left-0 border-right-0">
    <i class="material-icons align-middle">
      format_align_center
    </i>
  </button>
  <button class="p-2 btn btn-outline-dark">
    <i class="material-icons align-middle">
      format_align_right
    </i>
  </button>
</div>
<button class="m-1 btn btn-outline-dark rounded-0 border-dark">
  Follow
</button>
<button class="m-1 btn btn-sm btn-contained-dark elevation-2">
  Small
</button>

    Matilda provides .btn-rounded class to make circle buttons. It works well with Material icons.

    
<button class="m-1 btn btn-lg btn-contained-success elevation-8 btn-rounded">
  A
</button>
<button class="m-1 btn btn-contained-danger elevation-2 btn-rounded">
  <i class="material-icons">
    close
  </i>
</button>
