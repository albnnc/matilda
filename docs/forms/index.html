    Basics

    Basic form controls don't have any complex styling, borders or floating labels. It makes easy to use them as building blocks while creating other components.

    search
    
<input class="form-control"
       placeholder="Basic .form-control"
>
<div class="mt-2 px-2 card">
  <input class="form-control form-control-sm"
         placeholder="Any questions?"
  >
</div>
<div class="mt-3 px-2 card">
  <div class="input-group">
    <div class="input-group-prepend">
      <div class="input-group-text pl-1">
        <i class="material-icons text-muted">
          search
        </i>
      </div>
    </div>
    <input class="form-control form-control-lg"
           placeholder="Search"
    >
  </div>
</div>

    Materialized

    One can add simple accent line and line ripple by adding .form-underlineelement. You don't need additional classes for small or large inputs.

    
<div class="form-row">
  <div class="col-lg mb-3 mb-lg-0">
    <input class="form-control"
           placeholder="Input with a line under"
    >
    <div class="form-underline">
    </div>
  </div>
  <div class="col-lg">
    <input class="form-control"
           placeholder="Disabled input"
           disabled
    >
    <div class="form-underline">
    </div>
  </div>
</div>

    Add a custom styled label for the float label element with a.form-float-container and .form-float elements. You can also use .form-float-container-filled class to make your text field look filled.
